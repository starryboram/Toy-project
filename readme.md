
## Project1. 자전거 수요 예측 프로젝트
* 프로젝트 기간: 2022. 4. 21. ~2022. 4. 25. (평일만 진행)
* 프로젝트 인원: 3명
* 프로젝트 목표: 다양한 데이터 전처리 시도 및 머신러닝 모델 학습
* Data feature: 대여 날짜, 시간, 요일, 계절, 날씨, 체감 온도 등
* 평가지표: RMSLE
* 활용한 데이터셋: https://www.kaggle.com/c/bike-sharing-demand
* 데이터 처리내역 시각화
